## Developing Locally

To clone and set up this repository, use the following commands:

```bash
git clone https://github.com/DaQMinh/nextjs-starter.git
cd nextjs-starter
```

## Getting Started
First, Install dependencies and then run dev script
```bash
yarn install
yarn dev
```
