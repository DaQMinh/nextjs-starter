## Developing Locally

To clone and set up this repository, use the following commands:

```bash
git clone https://github.com/DaQMinh/nextjs-starter.git
cd nextjs-starter
```

## Getting Started
First, Install package
```bash
npm install
# or
yarn install
```
then, run the development server:

```bash
npm run dev
# or
yarn dev
```
